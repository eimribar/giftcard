# Agentss Gift Card Booking Page

Standalone landing page for the $50 gift card demo booking offer.

## Features

- Dark HUD design with orange corner accents
- Full-screen grid texture
- Glassmorphic gift card selection (Amazon & Hotels.com)
- Direct Calendly integration
- Optimized for Vercel deployment

## Deployment

This page is deployed as a standalone static site on Vercel.

## Local Development

Simply open `index.html` in a browser or run a local server:

```bash
python3 -m http.server 3000
```
